# 🧹 Data Cleaning Practice Hub

This repository is a dedicated space for tracking my progress in **Data Cleaning and Wrangling**. The sole focus here is taking messy, raw data and fixing it—handling errors, managing missing information, and ensuring data quality. 

No machine learning, no complex visualizations—just pure data cleaning workflows.

---

## 🛠️ Core Focus Areas

This repository tracks my hands-on practice across essential data cleaning tasks:

* **Missing Data:** Identifying null values, dropping incomplete records, and filling gaps using strategic imputation.
* **Duplicates:** Locating and removing redundant rows or features to maintain unique records.
* **Data Type Conversions:** Fixing wrongly formatted data (e.g., converting text strings to proper date-time objects or numeric types).
* **Structural Errors & Typos:** Standardizing text columns, stripping unwanted whitespace, and fixing inconsistent naming conventions.
* **Outlier Filtering:** Identifying and removing extreme anomalies or data entry errors that skew the data.
